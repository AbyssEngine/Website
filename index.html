Abyss Engine

The ARPG game engine

AbyssEngine is a game engine designed to run games similar to 2000's style ARPGs such as Diablo II.
This repository provides the core engine AbyssEngine Engine, as well as the AbyssEngine Toolset. It also includes the library libAbyss, which is used by this project, and can be used by 3rd party applications as well.
NOTE: This is just the engine core. It will not run by itself.
